# התקנה והפעלה במחשב חדש — Hipis Flight Search

מדריך מלא להעמדת הפרויקט מאפס על מחשב חדש (Windows). הזמן המשוער: ~10 דקות.

---

## מה הפרויקט הזה
חיפוש כרטיסי טיסה תלת‑לשוני (עברית/אנגלית/רוסית) עם מחירים אמיתיים מ‑Aviasales.
**שרת Node יחיד** (`server.js`) שעושה שני דברים יחד:
- מגיש את האתר (`index.html`)
- מגיש API מחירים (`/price`, `/offers`) מאותו מקור — בלי בעיות CORS

עובד גם מקומית (דאבל‑קליק על `הפעל.bat`) וגם בענן (Render/Railway וכו').

---

## שלב 1 — להתקין Node.js
1. להוריד מ‑https://nodejs.org את גרסת **LTS** (חובה Node 18 ומעלה).
2. להתקין (Next → Next → Finish).
3. לבדוק שהותקן — לפתוח **PowerShell** ולהריץ:
   ```powershell
   node -v
   ```
   צריך להופיע מספר גרסה כמו `v20.x` או גבוה יותר.

---

## שלב 2 — להתקין Git ולמשוך את הקוד
1. להוריד Git מ‑https://git-scm.com ולהתקין.
2. לבחור תיקייה למחשב (למשל `C:\Projects`) ולהריץ ב‑PowerShell:
   ```powershell
   cd C:\Projects
   git clone https://github.com/igal58/hipus.git
   cd hipus
   ```

> **התחברות ל‑GitHub:** בפעם הראשונה שתעשה `git push` ייפתח חלון התחברות לחשבון GitHub (igal58). מתחברים פעם אחת והטוקן נשמר.

---

## שלב 3 — ליצור את קובץ הסוד `proxy.config.json`
הקובץ הזה **לא נמצא ב‑GitHub בכוונה** (הוא מכיל את הטוקן הסודי, ומופיע ב‑`.gitignore`).
לכן במחשב חדש צריך ליצור אותו ידנית בתוך תיקיית הפרויקט.

צור קובץ בשם `proxy.config.json` עם המבנה הבא:
```json
{
  "token":    "<<הטוקן מ-Travelpayouts>>",
  "marker":   "742171",
  "currency": "ils",
  "port":     8787
}
```

**מאיפה לוקחים את ה‑token:**
- הדרך הקלה: להעתיק את הקובץ `proxy.config.json` מהמחשב הישן (יושב בתיקיית הפרויקט שם) ל‑USB/דרייב ולהדביק כאן.
- או: להתחבר ל‑https://www.travelpayouts.com → API tokens → להעתיק את ה‑token.
- ה‑`marker` (742171) הוא קוד השותף (affiliate) — לא סודי, אפשר להשאיר כמו שהוא.

> ⚠️ אסור לכתוב את הטוקן בתוך קובץ שעולה ל‑GitHub. הוא נשאר רק ב‑`proxy.config.json` המקומי.

---

## שלב 4 — להפעיל
**הדרך הפשוטה:** דאבל‑קליק על `הפעל.bat` בתיקיית הפרויקט.
ייפתח חלון שחור עם ההודעה:
```
A browser window will open at http://localhost:8787
```
פותחים דפדפן בכתובת **http://localhost:8787** — והאתר רץ.

**מ‑PowerShell (חלופי):**
```powershell
cd C:\Projects\hipus
node server.js
```
(`npm start` עושה אותו דבר.)

> חשוב: להשאיר את החלון השחור פתוח כל עוד משתמשים באתר. סגירת החלון = עצירת השרת.

## פתרון תקלות נפוצות
- **`node` לא מזוהה** → לא הותקן Node, או צריך לסגור ולפתוח מחדש את PowerShell אחרי ההתקנה.
- **האתר עולה אבל אין מחירים אמיתיים** → חסר/שגוי `proxy.config.json` (token). בדוק שהקובץ קיים בתיקייה ושה‑token תקין.
- **`git push` נכשל / מבקש סיסמה** → להתחבר מחדש לחשבון GitHub (igal58) בחלון ההתחברות.
- **הפורט 8787 תפוס** → לשנות `"port"` ב‑`proxy.config.json` למספר אחר (למשל 8080) ולפתוח את הדפדפן באותו פורט.
