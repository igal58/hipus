# Hipis — מסמך המשך (Handoff)

> מחר / במחשב אחר: פתח את Claude בתיקייה הזו ואמור לו **"תקרא את _המשך_מכאן.md"** — והוא יהיה מעודכן לגמרי.

## מה זה
אתר חיפוש כרטיסי טיסה (קובץ HTML יחיד + שרת Node קטן). עברית/אנגלית/רוסית, RTL, 6 ערכות צבע.
**מטרת המשתמש:** להיות טוב יותר מ-Google Flights בנישה (ישראל, רב-לשוני, השוואת אתרים, מבצעים, עמלות).

## איפה זה חי
- **ענן (חי):** https://hipus.onrender.com  (Render, שכבה חינמית — נרדם אחרי ~15 דק׳, מתעורר ~50ש)
- **קוד ב-GitHub:** https://github.com/igal58/hipus  (פרטי, ענף `main`; Render בונה אוטומטית בכל push)
- **מקומית:** התיקייה הזו. הרצה: לחיצה כפולה על `הפעל.bat` (צריך Node מותקן).

## קבצים עיקריים
- `index.html` — כל ה-UI + הלוגיקה (HTML+CSS+JS).
- `server.js` — שרת מאוחד: מגיש את הדף + API מחירים (`/price`, `/deals`, `/health`) מאותו origin (בלי CORS).
- `package.json` — להרצה בענן (`npm start`).
- `proxy.config.json` — **סודי, לא ב-GitHub**: מכיל את ה-API token + marker. חובה להעתיק ידנית!
- `הפעל.bat` — מפעיל מקומי. `proxy.js`/`start_proxy.bat` — ישנים, לא בשימוש.
- `Backup/` — גיבויים עם חותמת זמן (גיבוי לפני כל שינוי).

## מפתחות / חשבונות
- **Travelpayouts** (מקור המחירים): token שמור ב-`proxy.config.json` (מקומי) וב-Render כ-`TP_TOKEN`. marker = `742171` (אפיליאט). ⚠️ ה-token הוא 32 תווים ומתחיל ב-`4a34`.
- בענן ה-token/marker מוגדרים כ-Environment Variables ב-Render (לא בקוד).

## מה כבר עובד (הושלם)
1. **קישורים חיים** ל-Google/Skyscanner/Kiwi/Kayak/Momondo — חיפוש אמיתי למסלול+תאריכים.
2. **~233 ערים** ב-3 שפות; חיפוש חוצה-שפות + IATA.
3. **מחירים אמיתיים** דרך Travelpayouts (`market=il` כדי שיעבוד בענן), מוצגים בסיכום + על כל כרטיס.
4. **כרטיס Aviasales** "הזול ביותר": מחיר + עצירות + משך + כפתור רכישה עם ה-marker (עמלה).
5. **עצירות + משך על כל שורה**.
6. **טיסות סודיות / מבצעים חמים** (`/deals`): הזול מתל אביב לכל העולם; לחיצה ממלאת חיפוש.
7. **בורר נוסעים** (מבוגרים/ילדים/תינוקות) — עובר לכל הקישורים; מחיר "לנוסע".
8. **בורר כבודה** (יד/רשומה) — מועבר ל-Google; מסומן "בתשלום נפרד בהזמנה" (אין מחיר כבודה אמיתי — אין לזה מקור חינמי).
9. **שורת כלים יציבה** בהחלפת שפה (`direction:ltr`).

## עקרונות עבודה (חשוב!)
- **גיבוי לפני כל שינוי** ל-`Backup/` (חותמת זמן).
- מבצעים שינוי **רק אחרי שהמשתמש כותב "תתחיל"**.
- **יושר מעל הכול:** לא להמציא נתונים. מחיר/עצירות/משך מגיעים ממקור אחד (Aviasales) — מסומנים "מהזול שנמצא", לא מיוחסים לכל מנוע בנפרד.

## איך ממשיכים לפתח
1. הרץ מקומית (`הפעל.bat`), ערוך `index.html`/`server.js`.
2. דחיפה לענן: `git add -A` → `git commit -m "..."` → `git push origin main`. Render בונה אוטומטית (~2 דק׳).
3. אימות: בדוק https://hipus.onrender.com/health ו-`/price?origin=TLV&destination=ATH&depart=2026-07-01&return=2026-07-08`.
